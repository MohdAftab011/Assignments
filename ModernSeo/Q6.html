HTML vs HTML5

HTML HTML5
It required plugins like Adobe Flash to support audio and video content. Provides built-in support for multimedia elements such as video and audio without the need for a plugin.
It has fewer elements as compared to HTML5. Includes new elements and form attributes such as <time>, <date>, and <color>. Also supports attributes like required and autofocus for input elements.
It does not have support for local storage. It has support for local storage using the localStorage API.
Less semantic elements, thereby providing fewer web accessibility features. Supports more semantic elements such as <header>, <footer>, <nav>, and <article>, thereby improving accessibility.
